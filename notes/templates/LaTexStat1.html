Оформление отчёта в LaTeX

Где и как писать отчёт?

Для написания отчёта лучше использовать программу texstudio, которую можно скачать с официального сайта. Это довольно удобная среда, в которой уже заложено много функций в виде иконок, так что, особо не напрягаясь, уже можно красиво оформить статью или отчёт.

Отличное руководство по вёрстке в системе LaTeX - это книжка Львовского С.М. (предлагаем найти книжку в интернете самостоятельно)

По сути LaTeX это как отдельный язык программирования, поэтому к процедуре написания текста стоит относиться как к оформлению своеобразного кода (в виде блоков, преамбул и т.п.). Важно отметить, что любая команда в техе начинается с обратного слеша \.... Для начала нужно создать файл расширения .tex и задать начальные параметры документа. Мы будем сразу описывать основные моменты на примере отчёта с физического практикума. Основной блок (начало и конец документа) задаётся ключевыми словами \begin{document}...\end{document}. Так задаётся каждый отдельный блок или структура в LaTeX. В начале любого документа импортируются все нужные пакеты и задаются параметры документа: 

\documentclass[a4paper, 12pt]{article}    % Документ формата А4, размер шрифта 12pt

\usepackage[utf8]{inputenc}    % Задание кодировки
\usepackage[english,russian]{babel}   % Поддержка русского и английского языка
\usepackage{cmap}     % Правильная поддержка PDF документов и правильная кодировка символов
\usepackage[T2A]{fontenc}    % Дополнительный текстовый процессор
\usepackage{graphicx}     % Библиотека для включения рисунков и графиков \includegraphics
\usepackage[intlimits]{amsmath}   % Адаптация пределов для интегрирования с математическими формулами
\usepackage{amssymb}
\usepackage{wrapfig}   % Библиотека для вставки картинок параллельно с текстом
\usepackage[table,xcdraw]{xcolor}    % Библиотека для закраски табличек
\usepackage{indentfirst}   % Добавляется сразу красную строку в тексте

\usepackage{babel,blindtext}   % Фиксим постановку графиков и подписец
\usepackage{float}

\linespread{1.2}    %  Расстояние между строчками
\oddsidemargin=-18pt      % Задают отступы для листа (границы размещения текста)
\topmargin = -24pt
\textwidth = 484pt
\textheight = 678pt

Важной частью красивого оформления являются колонтитулы, которые реализуются с помощью пакета fancyhdr. На самом деле он содержит много функций, которые можно загуглить в интернете. Вполне легко сообразить их назначения по названию методов, и лучше понимать аббревиатуры параметров:

E: Even page Чётная страница
O: Odd page Нечётная страница
L: Left field Левое поле
C: Center field Центральное поле
R: Right field Правое поле
H: Header Заголовок
F: Footer Нижний колонтитул

Параметры задаются так же до начала документа: 

\usepackage{fancyhdr}
\pagestyle{fancy}
% этим мы убеждаемся, что заголовки глав и 
% разделов используют нижний регистр.

\fancyfoot{}
\fancyfoot[CF]{\bfseries\thepage}   % Нумерация страниц по центру

\fancyhead[L]{\textit{Опыт какой-то там}} %  Надпись сверху-слева у колонтитула
\fancyhead[RE]{\bfseries\leftmark}
\fancyhead[R]{Горюнов О.А., Плешков Д.С., Сиднев А.А.}   % Надпись сверху-справа у колонтитула
\renewcommand{\headrulewidth}{0.5pt}
\renewcommand{\footrulewidth}{0pt}
\addtolength{\headheight}{0pt} % оставляем место для линейки
\fancypagestyle{plain}{%
	\fancyhead{} % на обычных страницах убираем колонтитулы
	\renewcommand{\headrulewidth}{0pt} % и линейку
}

\setlength{\parskip}{0.2cm}

Титульный лист

Дальше мы начинаем оформлять сам документ. Обычно не обращают особого внимания на оформление титульного листа с содержанием, но строго говоря, это важно в каких-то документах и статьях. Поэтому, необходимые размеры и отступы всегда смотрите в правилах оформления того издателя, для которого вы пишете свой отчёт. Мы приведём пример нашего титульного листа, с которым написаны наши лабораторные работы на сайте.

		\hfill \break

		\hfill \break
		\hfill \break
		\hfill\break

		\large{\textbf{Опыт какой-то там}}\\
	\end{center}


	\hfill \break

		\thispagestyle{empty} % выключаем отображение номера для этой страницы

		% КОНЕЦ ТИТУЛЬНОГО ЛИСТА

\hfill \break используется для создания пустых строк. Так с помощью них можно подогнать отступы для текста. Далее командой \tableofcontents создаётся лист с содержанием документа, и командой \clearpage переходит на чистый лист. Далее зависит всё от вашей задачи. Мы лишь отметим основные моменты, которые нам пригодились при оформлении отчётов. Во-первых, это добавление секции в содержание без нумерации. На примере оформления протокола: 

{\centering\subsection*{Протокол} }
\addcontentsline{toc}{subsection}{\textbf{Протокол}}{

\centering отвечает за отцентровку текста, поэтому его при необходимости можно просто убрать. Важно понимать как идёт нумерация секций и подсекций, для этого просто можно самим попробовать их создавать командой \section, для подсекции объявляется ключевое слово sub (\subsection, \subsubsection, ...). Причём лучше ознакомиться с типами документов. В article нумерация начинается с единицы, а например у report с нуля. Но отсчёт нумерации всегда можно изменить, найдя соответствующую команду.

Списки

Иногда нужно создать список. Пример создания нумерованного списка: 

\begin{enumerate}
  \item первый элемент первого уровня содержит список
    \begin{enumerate}
        \item элемент списка второго уровня
        \item второй элемент списка второго уровня
    \end{enumerate}
  \item  второй элемент первого уровня
\end{enumerate}

Пример создания маркированного списка: 

\begin{itemize}
  \item Level 0 Item 0 (первый уровень)
  \item Level 0 Item 1 (первый уровень)
  \begin{itemize}
    \item Level 1 Item
    \begin{itemize}
      \item Level 2 Item
      \begin{itemize}
        \item Level 3 Item
      \end{itemize}
    \end{itemize}
  \end{itemize}
\end{itemize}

Вставка картинки/графика после текста

На самом деле картинки можно рисовать и в самом техе, но с использованием специальной библиотеки tikz. Можно и просто импортировать график или рисунок как картинку. Только важно сохранять всё в одной папке с проектом (можно создать подпапку с проектом для удобства). Для этого используют \begin{figure}[]...\end{figure}; просто создаётся отдельное пространство для фигуры, причём в квадратных скобках можно указать параметры позиционирования (например h перенесёт фигуру на новую страницу). Пример кода: 

\begin{figure}[h]
	\centering
	\includegraphics[width=0.65\linewidth]{image}   % width = размер (0 - 1)
	\caption{Подпись}
	\label{fig:parres}
\end{figure}

Вставка картинки параллельно с текстом

Аналогично можно вставить изображение и вместе с текстом. Для этого используется команда \begin{wrapfigure}...\end{wrapfigure}. Однако нужно учитывать, что картинка хорошо вставляется, когда она полностью заполнена текстом, иначе придётся подправлять, используя \hfill \break. Пример реализации вместе с текстом: 

\begin{wrapfigure}{l}{0.5\textwidth}
	\includegraphics[width=0.45\textwidth]{ris3}
	\caption{Силовые линии электрического поля и эквипотенциальные поверхности между электродами лампы в плоской модели.}
	\label{}
\end{wrapfigure}

	Энергия электронов, испытывающих соударения в различных точках слоя, могут различаться на величину порядка
	\[\Delta W_\lambda=e\lambda\frac{d\varphi}{dn},\]
	где $\frac{d\varphi}{dn}$ - производная потенциала по нормали к $S_1$.

	Разброс по энергиям на $\Delta W_\lambda$ расширяет максимум анодно-сеточной характеристики и уменьшает крутизну падающего участка.


\hfill \break

	Проще всего это понять на "плоской модели" \ со сплошной сеткой, проницаемой для электронов. В такой модели электрические поля между катодом и сеткой, сеткой и анодом однородны, а эквипотенциальные поверхности, показанные на рис. 3 штриховыми линиями, представляют собой плоскости, параллельные электродам.

Там так же есть несколько параметров {r} - вставка картинки справа на странице ({l} - слева). Все числовые параметры задают размер изображения и размер области, занимаемой фигурой.

Две картинки рядом

Реализуется аналогично вставке обычного изображения. Только для фигуры используем minipage: 

	\begin{center}
		\begin{figure}[H]
			\begin{minipage}{.5\textwidth}
				\centering
				\includegraphics[width=.95\linewidth]{ris1}
				\caption{Принципиальная схема экспериментальной установки.}
				\label{fig:parres}
			\end{minipage}
			\begin{minipage}{.5\textwidth}
				\centering
				\includegraphics[width=.95\linewidth]{ris2}
				\caption{Анодно-сеточная характеристика газонаполненной лампы.}
				\label{fig:parres}
			\end{minipage}
		\end{figure}
	\end{center}

Формулы

Немного уделим внимание написанию формул. Это реализуется так же с помощью специальных команд, и сначала может показаться это сложным и непонятным. Но спустя время тренировок, вы сможете спокойно их набирать как обычный текст. Лучше почитать про оформление формул у Львовского, а для математических символов воспользоваться следующим сайтом.

Если формула используется вместе с текстом, то лучше использовать $...$, где на месте троеточия пишется формула. Так формула никуда не съезжает и подгоняет свой размер под текст. Для выделения формулы отдельно используют \[...\]. А для создания маркировки формулы также добавляется внутри \eqno(1.1). Существует много команд под разные нужды, лучше просто поискать в интернете.

Пример кода использования формул приведён ниже: 

После прохождения пластинки $\lambda/4$ возникает эллиптическая поляризация, так как пластинка повёрнута на $45^\circ$ (т.е. находится в диагональном положении) можно показать на основе формулы (7), что интенсивность не будет меняться:
	\[\text{Если} \ \ \Delta n d=\frac{\lambda}{4} \ \ \ \Rightarrow \ \ \ \frac{\delta}{2} = \frac{\pi}{4} \ \ \ \Rightarrow \ \ \ sin^2(\frac{\delta}{2})=\frac{1}{2}\]
	\[\text{В диагональном положении } \alpha=\frac{\pi}{4} \ \ \ \Rightarrow \ \ \ \sin2\alpha = 1\]
	\[I=I_o[cos^2(\beta - \frac{\pi}{4})-\frac{1}{2}sin2\beta]\]
	\[cos^2(\beta - \frac{\pi}{4})=\frac{1+cos(2\beta-\pi/2)}{2}\]
	\[\Rightarrow \ \ I=I_o\Bigl[\frac{1}{2}+\frac{1}{2}sin2\beta-\frac{1}{2}sin2\beta\Bigr]=\frac{I_o}{2}\]
	Учтём, что изначально шёл поляризованный свет, тогда $I=I_\text{ист}/4$.
	То есть не зависит от поворота анализатора (угла $\beta$).
